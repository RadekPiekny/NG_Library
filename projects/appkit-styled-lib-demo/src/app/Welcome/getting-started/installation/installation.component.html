Template npmrc Creation of auth files

Once you've created your Angular project, it's essential to establish a template for the project's .npmrc file. This template, named template.npmrc, should reside in the root of your Angular application.


Remember, the template.npmrc file serves as a blueprint for other developers working on the project. It allows them to create their own .npmrc file, which is necessary for library downloads.

It's crucial to include the template.npmrc in your Git repository so that everyone has access to it.

Dark mode Light settings from the library

Light mode should be OS driven first only then from interaction in the application iteself.

The library provide a service that would automatically detect change in OS settings and change the application accordingly

The service can provide access to lightmode observable and method to change if the app would have custom switcher


In order to use dark/light mode in the application it is required to:


  1. update body
  2. Update app.module.ts
  3. Update app.component.ts
  4. Update app.component.html
Dynamic theme Light settings from the library

The library offers the capability to manually switch themes using a dedicated service
