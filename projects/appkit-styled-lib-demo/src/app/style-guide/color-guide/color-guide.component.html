How to use

The library use Tailwind Css framework hence all the tailwind functionality regarding the color will work based on the usage of the framework. The colors are both in CSS variables which you can observe in the inspect and in all variations tailwind provides.

Here are some CSS classes examples:
    • bg-primary-200
    • text-primary
    • border-teal-600
    • border-b-teal-300
    • border-t-teal-300
    • bg-primary
    • border-primary-200
